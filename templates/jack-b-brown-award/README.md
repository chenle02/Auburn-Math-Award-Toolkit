# Jack B. Brown Award Nomination Package Template

**Jack B. Brown Endowed Faculty Award**

## Checklist

- [ ] Update `\def\Nominee{...}` in `Nomination_Package_JBBrown.tex`
- [ ] Update `\def\Award{YYYY--YYYY Jack B. Brown Endowed Faculty Award}` with years
- [ ] Replace `Nomination_Letter.pdf` with the actual nomination letter (signed PDF)
  - Must address both teaching excellence AND research accomplishments
- [ ] Replace `CV.pdf` with the nominee's current CV
- [ ] Replace `Research_Statement.pdf` with a 1--3 page research statement
- [ ] Replace `Teaching_Evals_YYYY.pdf` files with actual evaluations (past 3--5 years)
- [ ] Replace `Support_Letter_Faculty.pdf` with a faculty colleague's letter
- [ ] Replace `Support_Letter_Student.pdf` with a student support letter
- [ ] Update section/subsection titles in the `.tex` file to match actual courses
- [ ] Compile twice: `pdflatex Nomination_Package_JBBrown.tex && pdflatex Nomination_Package_JBBrown.tex`
- [ ] Review the output PDF for correctness
- [ ] Submit to the Department Chair by **April 1**

## Eligibility Reminder

1. Assistant, associate, or full professor
2. Must personify Dr. Jack B. Brown
3. Strong commitment to students
4. Excellence in **both** teaching and research

## Selection Committee

- Previous award recipient
- Graduate Program Officer (GPO)
- Chair of the Undergraduate Studies Committee
- Final approval: Dean of COSAM
